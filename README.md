# online-tests
University project.

Project requires `online_tester_service` schema to exist.
To run locally, you need to initialise env database variables:

    ENV=dev
    PORT=8080
    HOST=yourHost
    USER=yourUsername
    PASSWORD=yourPassword
    DBNAME=online_tester_service
    DBPORT=5432
